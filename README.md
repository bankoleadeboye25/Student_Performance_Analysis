# 📊 Student Performance Analysis (JSS1)

This project analyzes mathematics performance of JSS1 students using Python. The goal is to explore score distribution, identify trends, and generate insights from real-world educational data.

## 🔧 Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## 📈 Analysis Performed
- Summary statistics (mean, distribution)
- Histogram visualization
- Boxplot analysis

## 🔍 Key Findings
- Average score ≈ 54, indicating moderate performance
- Most students scored between 50–60
- Presence of both low and high outliers
- Performance is concentrated within a narrow range

## 📁 Files
- `student_scores.csv` → Dataset
- `student_analysis.ipynb` → Python analysis
- `student_performance_report.pdf` → Report
